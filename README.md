# Кэшбэк Ассистент

PWA-приложение для управления категориями кэшбэка банковских карт. Работает офлайн, устанавливается на главный экран как нативное приложение.

## Возможности

- Импорт данных о кэшбэке в формате **COIN 0.9.0** (файл, текст или URL)
- Импорт из скриншота через AI-чат (DeepSeek, ChatGPT, Gemini — промпт встроен)
- Автоматическое определение количества выбираемых категорий
- Рекомендации по оптимальному выбору категорий с учётом всех банков
- Категории «не в лимите» для постоянного кэшбэка (На все покупки и т.п.)
- Дополнительные категории (пользовательские и доп. % к существующим)
- Группировка синонимичных категорий (АЗС + Топливо = одна группа)
- Справочник MCC-кодов с поиском
- Напоминание в Google Календаре с настраиваемым днём (1–28)
- Баннер напоминания 1–5 числа каждого месяца
- Перестановка банков в списке (↑ ↓)
- Копирование выбора с прошлого месяца
- Светлая и тёмная тема
- Полная офлайн-работа (Service Worker)
- Экспорт/импорт бэкапа (с настройками: тема, день напоминания)
- Обновление банка из нового COIN файла (с сохранением или сбросом выбора)

## Установка

1. Откройте [nuke79.github.io/fcashback](https://nuke79.github.io/fcashback) в Chrome / Edge на Android
2. Нажмите «Добавить на главный экран» (или кнопку установки в шапке)
3. Приложение работает как нативное — без адресной строки, с доступом офлайн

## Структура приложения

### Вкладка «Выбранное»

Все выбранные категории по всем банкам в одном списке — что именно выбрано, % кэшбэка, лимит, банк-источник.

### Вкладка «Банки»

Карточки банков с цветовой индикацией:
- **Жёлтая рамка** — категории не выбраны (кнопка «Выбрать»)
- **Цветная рамка** — есть выбранные категории
- **Яркая рамка** — список категорий раскрыт

При разворачивании — список категорий с кнопками: Рекомендации, Прошлый, Доп., Месяц.

### Вкладка «MCC»

Справочник Merchant Category Code — 4-значные коды, по которым банк определяет категорию покупки. Поиск по коду или названию.

## Напоминание

Приложение может создать ежемесячное событие в Google Календаре:
- День напоминания настраивается: **Справка → Ещё → День напоминания** (1–28)
- При смене дня — старое событие нужно удалить вручную, кнопка «Обновить в календаре» создаст новое
- Настройка сохраняется в бэкапе и переносится между устройствами

## Формат COIN 0.9.0

Приложение использует открытый стандарт **COIN** (Cashback Operations INterchange) для обмена данными о кэшбэке.

- Спецификация: [gitverse.ru/goodman/coin](https://gitverse.ru/goodman/coin/content/master/schema)
