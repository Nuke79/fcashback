# Кэшбэк Ассистент

PWA-приложение для управления категориями кэшбэка банковских карт. Работает офлайн, устанавливается на главный экран.

## Возможности

- Импорт данных о кэшбэке в формате **COIN 0.9.0** (файл или JSON-текст)
- Импорт из скриншота через AI-чат (протокол и промпт встроены в приложение)
- Автоматическое определение количества выбираемых категорий из описаний
- Рекомендации по оптимальному выбору категорий с учётом всех банков
- Группировка синонимичных категорий (АЗС + Топливо и АЗС = одна группа)
- Справочник MCC-кодов с поиском
- Светлая и тёмная тема
- Полное офлайн-работа через Service Worker
- Экспорт/импорт бэкапа данных
- Копирование выбора с прошлого месяца

## Формат COIN 0.9.0

Приложение использует открытый стандарт **COIN** (Cashback Operations INterchange) для обмена данными о кэшбэке.

- Спецификация: [gitverse.ru/goodman/coin](https://gitverse.ru/goodman/coin/content/master/schema)

## Установка

1. Откройте [nuke79.github.io/fcashback](https://nuke79.github.io/fcashback) в Chrome/Edge на Android
2. Нажмите «Добавить на главный экран» (или кнопку установки в шапке приложения)
3. Приложение работает как нативное — без адресной строки, с доступом офлайн

## Структура проекта

```
├── index.html        # Всё приложение (HTML + CSS + JS)
├── sw.js             # Service Worker (кеширование, офлайн)
├── manifest.json     # PWA-манифест
└── icon-192.png      # Иконка приложения
```

## Сборка

Для обновления версии:

1. Внести изменения в `index.html`
2. Увеличить версию в `sw.js`: `const CACHE_NAME = 'cashback-vXX';`
3. Запаковать: `zip -j fcashback-vXX.zip index.html sw.js manifest.json icon-192.png`
